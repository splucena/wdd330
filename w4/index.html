Object Oriented Programming in JavaScript

Object Oriented Programming

Object-oriented programming is often used to model representations of object in the real world.

Encapsulation

This involves keeping all programming logic inside an object and methods available to implement functionality, without the outisde world needing to know how it's done.

Polymorphism

This means various objects can share the same method, but also have the ability to override shared methods with a more specific implementation.

Polymorphism means objects are able to override methods with a more specific implementation.

Inheritance

This means we can an object that already exist and inherit all its properties and methods. We can improve on its functionality by adding new properties and methods.

ES6 Class Declaration

ES6 introduced the new class declaration syntax that does exactly the same things as a constructor function.

Static Methods

A static method is called by the class directly rather than by instances of the class. Also called class methods in other programming languages.

Public and Private Methods

By defaulat, an object's method are public in JavaScript.

We can use the concept of variable scope to keep some properties and methods private inside a class definition
