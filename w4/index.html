Object Oriented Programming in JavaScript

Object Oriented Programming

Object-oriented programming is often used to model representations of object in the real world.

Encapsulation

This involves keeping all programming logic inside an object and methods available to implement functionality, without the outisde world needing to know how it's done.

Ploymorphism

This means various objects can share the same method, but also have the ability to override shared methods with a more specific implementation.
