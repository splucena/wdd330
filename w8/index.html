CSS3 Transforms and Transitions

Lets you translate, rotate, scale, and/or skew any element on the page.

Translate

We can manipulate the element's appearance using transform functions. The value of the transform property is one or more transform functions (separated by spaces) that will be applied in the order they're provided.

Scaling

Scales an element by the defined factors horizontally then vertically.

Rotation

The rotate() function rotates an element around the point of origin by a specified angle value. Default point of origin is the element's center.
